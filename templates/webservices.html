Webservices page


Most Mutalyzer functionality is available trough a SOAP webservice. A WSDL description is available for easy generation of client programs in many languages.

Example clients

The following are some example client programs for the webservice. They use the checkSyntax method to determine if a variant description adheres to the HGVS format.
  • Python client using the suds library: client-suds.py
  • Python client using the soappy library: client-soappy.py
  • Ruby client using the Savon library: client-savon.rb
  • Mono client in C#: client-mono.cs
  • PHP client using the PHP5 SOAP extension: client-php.php
