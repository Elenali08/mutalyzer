Welcome to the Mutalyzer web site!

Mutalyzer aims to support checks of sequence variant nomenclature according to the guidelines of the Human Genome Variation Society.

Mutalyzer has different interfaces to collect the information necessary for the checks:

Mutalyzer's Name Checker takes the complete sequence variant description as input and checks whether it is correct.

 

Mutalyzer's GenBank Uploader allows you to upload and use your own reference sequences for checks of sequence variant descriptions. For this, you will need a text file in GenBank format. Alternatively, you may specify the HGNC gene symbol or the chromosomal range corresponding with the sequence to be uploaded.

 

The Mutalyzer Engine performing the checks is Python-powered. The Python Scripting Language was chosen for good readability and therefore easy maintenance.
